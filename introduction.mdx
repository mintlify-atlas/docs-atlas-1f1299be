# Welcome to AnyCarousel

AnyCarousel is a powerful, lightweight carousel component for React that handles real-world use cases like custom content, snapping, auto-scroll, and responsive layouts. Whether you're building an image gallery, content slider, or testimonial carousel, AnyCarousel adapts to your needs.

## Key Features

- **Lightweight & Minimalistic** — Zero dependencies (except React), minimal footprint
- **Mobile & Tablet Friendly** — Smooth swipe gestures for horizontal scrolling
- **Content Agnostic** — Works with images, videos, and any custom React components
- **Custom Snapping** — Control how items snap into place (`start`, `center`, `end`, or `none`)
- **Auto Scroll** — Configurable auto-slide interval with loop support
- **Custom Easing** — Use cubic-bezier curves for advanced scroll animations
- **Theming** — Built-in light and dark themes
- **Desktop Support** — Shift + Scroll for horizontal navigation
- **Touchpad Support** — Two-finger scroll gesture supported

## Installation

Get started with AnyCarousel by installing it via your favorite package manager:

```bash npm

```

```bash yarn
yarn add react-any-carousel
```

```bash pnpm
pnpm add react-any-carousel
```

</CodeGroup>

## Quick Example

Here's a simple example to get you started:

```tsx
import { Carousel } from "react-any-carousel";

function App() {
  return (
    <Carousel scrollSnapType="center" scrollOffset={300} duration={600}>
      <div className="card">Item 1</div>
      <div className="card">Item 2</div>
      <div className="card">Item 3</div>
    </Carousel>
  );
}
```
