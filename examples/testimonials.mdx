# Testimonials Carousel

Build beautiful testimonial carousels that automatically rotate through customer reviews with smooth transitions. Perfect for showcasing social proof on landing pages, product pages, or about sections.

## Basic Testimonial Carousel

A simple testimonial carousel with center snapping:

```tsx
import { Carousel } from "react-any-carousel";

function TestimonialCarousel() {
  const testimonials = [
    {
      name: "Sarah Johnson",
      role: "Product Manager",
      company: "TechCorp",
      content: "This product has transformed how our team collaborates. The intuitive interface and powerful features make it indispensable.",
      avatar: "https://i.pravatar.cc/150?img=1",
    },
    {
      name: "Michael Chen",
      role: "CEO",
      company: "StartupXYZ",
      content: "We've seen a 40% increase in productivity since implementing this solution. Highly recommended for any growing business.",
      avatar: "https://i.pravatar.cc/150?img=2",
    },
    {
      name: "Emily Rodriguez",
      role: "Designer",
      company: "Creative Co",
      content: "As a designer, I appreciate the attention to detail. The user experience is seamless and the design is beautiful.",
      avatar: "https://i.pravatar.cc/150?img=3",
    },
  ];

  return (
    <Carousel
      scrollSnapType="center"
      scrollOffset={600}
      duration={800}
    >
      {testimonials.map((testimonial, index) => (
        <div
          key={index}
          style={{
            width: "500px",
            padding: "40px",
            margin: "0 30px",
            background: "white",
            borderRadius: "16px",
            boxShadow: "0 4px 16px rgba(0, 0, 0, 0.1)",
            flexShrink: 0,
          }}
        >
          <div style={{ marginBottom: "24px", fontSize: "32px", color: "#e5e7eb" }}>"</div>
          <p style={{ margin: "0 0 24px 0", fontSize: "16px", lineHeight: "1.6", color: "#374151" }}>
            {testimonial.content}
          </p>
          <div style={{ display: "flex", alignItems: "center", gap: "16px" }}>
            <img
              src={testimonial.avatar}
              alt={testimonial.name}
              style={{ width: "56px", height: "56px", borderRadius: "50%" }}
            />
            <div>
              <div style={{ fontWeight: "600", fontSize: "16px", color: "#111827" }}>{testimonial.name}</div>
              <div style={{ fontSize: "14px", color: "#6b7280" }}>
                {testimonial.role} at {testimonial.company}
              </div>
            </div>
          </div>
        </div>
      ))}
    </Carousel>
  );
}
```

<Note>
  Center snapping (`scrollSnapType="center"`) ensures testimonials are always perfectly centered, creating a focused reading experience.
</Note>

## Auto-Scroll with Custom Easing

Create smooth automatic transitions between testimonials:

```tsx
import { Carousel } from "react-any-carousel";

function AutoScrollTestimonials() {
  const testimonials = [
    {
      name: "Alex Thompson",
      role: "Engineering Lead",

      rating: 5,
      avatar: "https://i.pravatar.cc/150?img=4",
    },
    {
      name: "Jessica Lee",
      role: "Marketing Director",
      content: "Our conversion rates improved by 35% after implementing this solution. The ROI speaks for itself.",
      rating: 5,
      avatar: "https://i.pravatar.cc/150?img=5",
    },
    {
      name: "David Kumar",
      role: "Founder",
      content: "Simple, powerful, and reliable. This is exactly what we needed to scale our operations.",
      rating: 5,
      avatar: "https://i.pravatar.cc/150?img=6",
    },
  ];

  return (
    <div style={{ maxWidth: "900px", margin: "0 auto", padding: "60px 20px" }}>
      <h2 style={{ textAlign: "center", marginBottom: "48px", fontSize: "36px", fontWeight: "700" }}>
        What Our Customers Say
      </h2>
      <Carousel
        scrollSnapType="center"
        scrollOffset={620}
        autoSlideInterval={5000} // Auto-advance every 5 seconds
        duration={1000}
        scrollEasing="cubic-bezier(0.25, 0.46, 0.45, 0.94)" // Smooth, natural easing
      >
        {testimonials.map((testimonial, index) => (
          <div
            key={index}
            style={{
              width: "560px",
              padding: "48px",
              margin: "0 30px",
              background: "linear-gradient(to bottom right, #ffffff, #f9fafb)",
              border: "1px solid #e5e7eb",
              borderRadius: "20px",
              boxShadow: "0 8px 24px rgba(0, 0, 0, 0.08)",
              flexShrink: 0,
              textAlign: "center",
            }}
          >
            <img
              src={testimonial.avatar}
              alt={testimonial.name}
              style={{
                width: "80px",
                height: "80px",
                borderRadius: "50%",
                marginBottom: "24px",
                border: "4px solid #2563eb",
              }}
            />
            <div style={{ marginBottom: "20px" }}>
              {Array.from({ length: testimonial.rating }).map((_, i) => (
                <span key={i} style={{ color: "#fbbf24", fontSize: "20px" }}>★</span>
              ))}
            </div>
            <p style={{
              margin: "0 0 24px 0",
              fontSize: "18px",
              lineHeight: "1.7",
              color: "#374151",
              fontStyle: "italic",
            }}>
              "{testimonial.content}"
            </p>
            <div style={{ fontWeight: "600", fontSize: "17px", color: "#111827", marginBottom: "4px" }}>
              {testimonial.name}
            </div>
            <div style={{ fontSize: "14px", color: "#6b7280" }}>{testimonial.role}</div>
          </div>
        ))}
      </Carousel>
    </div>
  );
}
```

<Tip>
  The easing function `cubic-bezier(0.25, 0.46, 0.45, 0.94)` creates a smooth, natural animation that feels polished and professional.
</Tip>

## Best Practices

### Auto-Scroll Timing

| Content Length | Recommended Interval |
|----------------|---------------------|
| Short (1-2 lines) | 3000-4000ms |
| Medium (3-4 lines) | 5000-6000ms |
| Long (5+ lines) | 7000-8000ms |

### Easing Functions for Testimonials

```tsx
// Smooth and elegant (recommended)
scrollEasing="cubic-bezier(0.25, 0.46, 0.45, 0.94)"

// Slightly bouncy (playful)
scrollEasing="cubic-bezier(0.34, 1.56, 0.64, 1)"

// Linear (simple)
scrollEasing="cubic-bezier(0, 0, 1, 1)"

// Ease-in-out (balanced)
scrollEasing="cubic-bezier(0.42, 0, 0.58, 1)"
```

### Accessibility Tips

<Warning>
  When using auto-scroll, always provide a pause mechanism (hover or explicit button) to ensure users have enough time to read content. This is important for accessibility.
</Warning>

```tsx
// Good: Provides pause on hover
<div
  onMouseEnter={() => setIsPaused(true)}
  onMouseLeave={() => setIsPaused(false)}
>
  <Carousel autoSlideInterval={isPaused ? undefined : 4000}>
    {/* testimonials */}
  </Carousel>
</div>

// Also good: Auto-scroll pauses when user clicks navigation
// This is built-in behavior in AnyCarousel
```

| Property | Value | Purpose |
|----------|-------|----------|
| `scrollSnapType` | `"center"` | Centers testimonials for focused reading |
| `autoSlideInterval` | `4000-6000` | Auto-advance every 4-6 seconds |
| `duration` | `800-1200` | Smooth, unhurried transitions |
| `scrollEasing` | `"cubic-bezier(0.25, 0.46, 0.45, 0.94)"` | Natural, professional motion |
| `scrollOffset` | Card width + margins | Ensures one testimonial scrolls at a time |
