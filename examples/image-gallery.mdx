# Image Gallery Carousel

Create a professional image gallery with smooth transitions, auto-scroll slideshow mode, and responsive image sizing. This example demonstrates center-snapping for perfect image alignment and custom navigation icons.

## Basic Image Gallery

A simple image gallery with center-aligned snapping:

```tsx
import { Carousel } from "react-any-carousel";

function ImageGallery() {
  const images = [
    "https://images.unsplash.com/photo-1506905925346-21bda4d32df4",
    "https://images.unsplash.com/photo-1469474968028-56623f02e42e",
    "https://images.unsplash.com/photo-1447752875215-b2761acb3c5d",
    "https://images.unsplash.com/photo-1433086966358-54859d0ed716",
    "https://images.unsplash.com/photo-1441974231531-c6227db76b6e",
  ];

  return (
    <div style={{ maxWidth: "1200px", margin: "0 auto" }}>
      <Carousel
        scrollSnapType="center"
        scrollOffset={600}
        duration={800}
      >
        {images.map((src, index) => (
          <img
            key={index}
            src={src}
            alt={`Gallery image ${index + 1}`}
            style={{
              width: "500px",
              height: "350px",
              objectFit: "cover",
              borderRadius: "12px",
              margin: "0 15px",
              flexShrink: 0,
            }}
          />
        ))}
      </Carousel>
    </div>
  );
}
```

<Note>
  The `scrollSnapType="center"` ensures images are always perfectly centered when scrolling stops.
</Note>

## Auto-Scroll Slideshow

Create an automatic slideshow with configurable intervals:

```tsx
import { Carousel } from "react-any-carousel";

function AutoScrollGallery() {
  const images = [
    { src: "https://images.unsplash.com/photo-1506905925346-21bda4d32df4", alt: "Mountain landscape" },
    { src: "https://images.unsplash.com/photo-1469474968028-56623f02e42e", alt: "Forest trail" },
    { src: "https://images.unsplash.com/photo-1447752875215-b2761acb3c5d", alt: "Desert sunset" },
    { src: "https://images.unsplash.com/photo-1433086966358-54859d0ed716", alt: "Ocean waves" },
    { src: "https://images.unsplash.com/photo-1441974231531-c6227db76b6e", alt: "Mountain peak" },
  ];

  return (
    <Carousel
      scrollSnapType="center"
      scrollOffset={600}
      autoSlideInterval={3000} // Auto-scroll every 3 seconds
      duration={1000}
      scrollEasing="cubic-bezier(0.25, 0.46, 0.45, 0.94)"
    >
      {images.map((image, index) => (
        <div
          key={index}
          style={{
            position: "relative",
            width: "550px",
            height: "400px",
            margin: "0 20px",
            flexShrink: 0,
          }}
        >
          <img
            src={image.src}
            alt={image.alt}
            style={{
              width: "100%",
              height: "100%",
              objectFit: "cover",
              borderRadius: "16px",
              boxShadow: "0 4px 12px rgba(0, 0, 0, 0.15)",
            }}
          />
          <div
            style={{
              position: "absolute",
              bottom: "20px",
              left: "20px",
              right: "20px",
              padding: "12px 20px",
              background: "rgba(0, 0, 0, 0.7)",
              backdropFilter: "blur(10px)",
              borderRadius: "8px",
              color: "white",
              fontSize: "14px",
            }}
          >
            {image.alt}
          </div>
        </div>
      ))}
    </Carousel>
  );
}
```

<Tip>
  Auto-scroll pauses automatically when users interact with the carousel by clicking navigation buttons.
</Tip>

## Responsive Image Sizing

Adapt image sizes based on screen width:

```tsx
import { Carousel } from "react-any-carousel";
import { useState, useEffect } from "react";

function ResponsiveGallery() {
  const [imageSize, setImageSize] = useState({
    width: 500,
    height: 350,
    scrollOffset: 520,
  });

  useEffect(() => {
    const updateSize = () => {
      const width = window.innerWidth;
      if (width < 640) {
        setImageSize({ width: 280, height: 200, scrollOffset: 300 });
      } else if (width < 1024) {
        setImageSize({ width: 400, height: 280, scrollOffset: 420 });
      } else {
        setImageSize({ width: 500, height: 350, scrollOffset: 520 });
      }
    };

    updateSize();
    window.addEventListener("resize", updateSize);
    return () => window.removeEventListener("resize", updateSize);
  }, []);

  const images = [
    "https://images.unsplash.com/photo-1506905925346-21bda4d32df4",
    "https://images.unsplash.com/photo-1469474968028-56623f02e42e",
    "https://images.unsplash.com/photo-1447752875215-b2761acb3c5d",
    "https://images.unsplash.com/photo-1433086966358-54859d0ed716",
  ];

  return (
    <Carousel
      scrollSnapType="center"
      scrollOffset={imageSize.scrollOffset}
      duration={800}
    >
      {images.map((src, index) => (
        <img
          key={index}
          src={src}
          alt={`Gallery image ${index + 1}`}
          style={{
            width: `${imageSize.width}px`,
            height: `${imageSize.height}px`,
            objectFit: "cover",
            borderRadius: "12px",
            margin: "0 10px",
            flexShrink: 0,
            transition: "width 0.3s ease, height 0.3s ease",
          }}
        />
      ))}
    </Carousel>
  );
}
```

<Warning>
  Make sure your `scrollOffset` is slightly larger than the image width to account for margins and ensure smooth scrolling.
</Warning>

## Complete Production Example

A full-featured image gallery with all features combined:

```tsx
import { Carousel } from "react-any-carousel";
import { useState } from "react";
import { FiChevronRight, FiPlay, FiPause } from "react-icons/fi";

function ProductionGallery() {
  const [autoScroll, setAutoScroll] = useState(true);

  const images = [
    { src: "https://images.unsplash.com/photo-1506905925346-21bda4d32df4", title: "Mountain Vista", location: "Rocky Mountains" },
    { src: "https://images.unsplash.com/photo-1469474968028-56623f02e42e", title: "Forest Path", location: "Pacific Northwest" },
    { src: "https://images.unsplash.com/photo-1447752875215-b2761acb3c5d", title: "Desert Sunset", location: "Arizona" },
    { src: "https://images.unsplash.com/photo-1433086966358-54859d0ed716", title: "Ocean Waves", location: "California Coast" },
    { src: "https://images.unsplash.com/photo-1441974231531-c6227db76b6e", title: "Alpine Peak", location: "Swiss Alps" },
  ];

  return (
    <div style={{ maxWidth: "1400px", margin: "0 auto", padding: "40px 20px" }}>
      <div style={{ display: "flex", justifyContent: "space-between", alignItems: "center", marginBottom: "24px" }}>
        <h2 style={{ margin: 0, fontSize: "24px", fontWeight: "600" }}>Nature Gallery</h2>
        <button
          onClick={() => setAutoScroll(!autoScroll)}
          style={{
            display: "flex",
            alignItems: "center",
            gap: "8px",
            padding: "8px 16px",
            background: "#2563eb",
            color: "white",
            border: "none",
            borderRadius: "8px",
            cursor: "pointer",
            fontSize: "14px",
          }}
        >
          {autoScroll ? <FiPause size={16} /> : <FiPlay size={16} />}
          {autoScroll ? "Pause" : "Play"}
        </button>
      </div>

      <Carousel
        scrollSnapType="center"
        scrollOffset={620}
        autoSlideInterval={autoScroll ? 4000 : undefined}
        duration={1200}
        scrollEasing="cubic-bezier(0.25, 0.46, 0.45, 0.94)"
        iconOptions={{
          icon: <FiChevronRight size={28} />,
          iconStyles: {
            color: "white",
            backgroundColor: "rgba(0, 0, 0, 0.6)",
            padding: "16px",
            borderRadius: "50%",
            backdropFilter: "blur(8px)",
            border: "2px solid rgba(255, 255, 255, 0.2)",
          },
        }}
      >
        {images.map((image, index) => (
          <div
            key={index}
            style={{
              position: "relative",
              width: "600px",
              height: "400px",
              margin: "0 10px",
              flexShrink: 0,
              borderRadius: "16px",
              overflow: "hidden",
              boxShadow: "0 8px 24px rgba(0, 0, 0, 0.12)",
            }}
          >
            <img
              src={image.src}
              alt={image.title}
              style={{
                width: "100%",
                height: "100%",
                objectFit: "cover",
              }}
            />
            <div
              style={{
                position: "absolute",
                bottom: 0,
                left: 0,
                right: 0,
                padding: "24px",
                background: "linear-gradient(to top, rgba(0, 0, 0, 0.8), transparent)",
                color: "white",
              }}
            >
              <h3 style={{ margin: "0 0 4px 0", fontSize: "20px", fontWeight: "600" }}>{image.title}</h3>
              <p style={{ margin: 0, fontSize: "14px", opacity: 0.9 }}>{image.location}</p>
            </div>
          </div>
        ))}
      </Carousel>
    </div>
  );
}
```

## Key Properties

| Property | Value | Purpose |
|----------|-------|----------|
| `scrollSnapType` | `"center"` | Centers images perfectly when scrolling stops |
| `scrollOffset` | `600` | Scrolls approximately one image width at a time |
| `autoSlideInterval` | `3000` | Auto-advances every 3 seconds (3000ms) |
| `duration` | `800-1200` | Smooth transition duration in milliseconds |
| `scrollEasing` | `"cubic-bezier(0.25, 0.46, 0.45, 0.94)"` | Custom easing for natural motion |
