This guide will help you create your first carousel component and understand the core concepts.

## Basic Example

<Steps>

<Step title="Import the Carousel component">

Start by importing the `Carousel` component from the package:

```tsx
import { Carousel } from 'react-any-carousel';
```

</Step>

<Step title="Add carousel items as children">

The Carousel component accepts any React elements as children. Each child becomes a carousel item:

```tsx
function App() {
  return (
    <Carousel>
      <div className="card">Item 1</div>
      <div className="card">Item 2</div>
      <div className="card">Item 3</div>
      <div className="card">Item 4</div>
    </Carousel>
  );
}
```

<Note>
  The `children` prop accepts any `ReactNode`, so you can use images, videos, custom components, or any JSX.
</Note>

</Step>

<Step title="Configure scroll behavior">

Add snap alignment and scroll offset for smooth navigation:

```tsx
<Carousel 
  scrollSnapType="center" 
  scrollOffset={300}
>
  <div className="card">Item 1</div>
  <div className="card">Item 2</div>
  <div className="card">Item 3</div>
  <div className="card">Item 4</div>
</Carousel>
```

**Props explained:**
- `scrollSnapType="center"` - Items snap to the center of the viewport. Options: `"start"`, `"center"`, `"end"`, or `"none"`
- `scrollOffset={300}` - Scrolls 300px when clicking navigation buttons. Set this to your item width for single-item scrolling.

</Step>

<Step title="Customize animation duration">

Control the scroll animation speed with the `duration` prop (in milliseconds):

```tsx
<Carousel 
  scrollSnapType="center" 
  scrollOffset={300}
  duration={600}
>
  <div className="card">Item 1</div>
  <div className="card">Item 2</div>
  <div className="card">Item 3</div>
</Carousel>
```

The `duration` prop accepts values between 100ms and 5000ms. Default is **600ms**.

</Step>

</Steps>

## Complete Working Example

Here's a full example with images and styling:

<CodeGroup>

```tsx App.tsx
import { Carousel } from 'react-any-carousel';
import './App.css';

function App() {
  const images = [
    'https://picsum.photos/400/300?random=1',
    'https://picsum.photos/400/300?random=2',
    'https://picsum.photos/400/300?random=3',
    'https://picsum.photos/400/300?random=4',
    'https://picsum.photos/400/300?random=5',
  ];

  return (
    <div className="app">
      <h1>Image Carousel</h1>
      <Carousel
        scrollSnapType="center"
        scrollOffset={420}
        duration={600}
        theme="light"
      >
        {images.map((src, index) => (
          <img
            key={index}
            src={src}
            alt={`Slide ${index + 1}`}
            className="carousel-image"
          />
        ))}
      </Carousel>
    </div>
  );
}

export default App;
```

```css App.css
.app {
  max-width: 1200px;
  margin: 0 auto;
  padding: 2rem;
}

.carousel-image {
  width: 400px;
  height: 300px;
  object-fit: cover;
  border-radius: 8px;
  margin: 0 10px;
}
```

</CodeGroup>

## Advanced Features

### Auto-Scroll Carousel

Make the carousel automatically scroll at fixed intervals:

```tsx
<Carousel
  scrollSnapType="center"
  scrollOffset={300}
  autoSlideInterval={3000}
>
  {/* Items will auto-advance every 3 seconds */}
  <div className="card">Item 1</div>
  <div className="card">Item 2</div>
  <div className="card">Item 3</div>
</Carousel>
```

<Tip>
  The `autoSlideInterval` prop accepts a duration in milliseconds. Auto-scroll stops when users interact with the carousel.
</Tip>

### Custom Easing Functions

Customize the scroll animation with CSS cubic-bezier easing:

```tsx
<Carousel
  scrollSnapType="start"
  scrollOffset={400}
  scrollEasing="cubic-bezier(0.25, 1, 0.5, 1)"
  duration={800}
>
  <div className="card">Item 1</div>
  <div className="card">Item 2</div>
  <div className="card">Item 3</div>
</Carousel>
```

The library uses the **bezier-easing** package to parse and apply custom easing functions. Default is `cubic-bezier(0.42, 0, 0.58, 1)`.

### Theme Support

Switch between light and dark themes for navigation icons:

```tsx
<Carousel theme="dark">
  <div className="card">Item 1</div>
  <div className="card">Item 2</div>
  <div className="card">Item 3</div>
</Carousel>
```

## Key Concepts

### Scroll Snap Type

Controls where items align when scrolling stops. Based on the CSS `scroll-snap-type` property:
- **`start`**: Items align to the start of the container
- **`center`**: Items align to the center (best for showcasing items)
- **`end`**: Items align to the end
- **`none`**: No snapping behavior

[Learn more about scroll-snap-type on MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/scroll-snap-type)

### Scroll Offset

Defines how far the carousel scrolls when navigation buttons are clicked:
- Set to **item width** to scroll one item at a time
- Set to **container width** to scroll a full page of items
- Must be a number representing pixels

<Tip>
  For single-item scrolling with 400px wide items and 20px gaps, use `scrollOffset={420}`.
</Tip>

## Browser Support

react-any-carousel works in all modern browsers that support:
- CSS Scroll Snap (all evergreen browsers)
- Touch events (mobile and tablet devices)
- RequestAnimationFrame API

### Desktop Navigation
- **Mouse wheel** + **Shift key**: Horizontal scrolling
- **Two-finger scroll** on trackpads: Native horizontal scrolling
- **Navigation buttons**: Click to scroll by `scrollOffset` amount

### Mobile Navigation
- **Touch gestures**: Swipe left/right to scroll
- **Navigation buttons**: Tap to scroll (when visible)
